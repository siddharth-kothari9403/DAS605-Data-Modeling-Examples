# DAS-605 Data Modeling Examples

This repository contains examples for various data models covered in the DAS-605 Data Modeling course.

1. xml-publisher - A full-stack application for creating, editing, and publishing news articles in multiple formats (XML, HTML, RSS, PDF) using XML as the core data format and XSL transformations. This allows us to do various kinds of Content Repurposing, which is used extensively in Content Publishing.
